1. 오늘 배운 것
   오늘은 JavaScript의 기본 문법과 DOM 조작에 대해 배웠다.
   변수 선언(let, const)과 함수 작성 방법을 익혔고, 버튼 클릭 시 숫자가 증가하는 카운터 기능을 구현했다.
   특히 document.getElementById()를 활용해 HTML 요소를 가져오고, textContent로 값을 변경하는 방식이 인상적이었다.

2. 어려웠던 점
   초기에는 변수 값이 변경되어도 화면에 바로 반영되지 않는 이유를 이해하기 어려웠다.
   또한 함수가 실행되는 시점과 이벤트(onclick)의 개념이 헷갈렸다.

3. 해결 방법
   직접 콘솔(console.log)을 찍어보면서 값이 어떻게 변하는지 확인했고,
   이벤트가 발생할 때 함수가 실행된다는 흐름을 반복 실습을 통해 이해했다.
   또한 간단한 카운터 앱을 여러 번 다시 만들어보면서 개념을 익혔다.

4. 느낀 점
   JavaScript는 단순히 문법을 외우는 것이 아니라,
   "HTML을 어떻게 조작하는지"를 이해하는 것이 중요하다는 것을 느꼈다.
   아직 완전히 익숙하지는 않지만, 반복해서 프로젝트를 만들어보면 자연스럽게 익힐 수 있을 것 같다.
