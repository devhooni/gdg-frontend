HTML의 기초를 배우는 과정은 웹 페이지가 어떻게 구성되고 동작하는지 이해하는 데 큰 도움이 되었다. 이번에 자기소개 페이지를 직접 만들어보면서 단순히 이론으로만 배우는 것이 아니라, 실제로 코드를 작성하고 결과를 확인하는 경험을 할 수 있었다. 특히 제목을 나타내는 h1 태그부터 문단을 구성하는 o 태그, 목록을 만들 때 사용하는 ul과 li 태그, 그리고 다른 페이지로 이동할 수 있게 해주는 a 태그 등 다양한 요소들을 직접 사용해보며 각각의 역할과 특징을 자연스럽게 익힐 수 있었다.

또한 HTML 문서의 기본 구조인 !doctype html 선언과 html, head, body 태그의 역할을 이해하면서 웹 페이지가 어떤 틀 안에서 구성되는지도 알게 되었다. head 영역에는 문서의 정보와 설정이 들어가고, body 영역에는 실제로 화면에 보이는 내용이 들어간다는 점을 실습을 통해 명확히 이해할 수 있었다. 이러한 구조를 기반으로 콘텐츠를 배치하면서 웹 페이지가 단순한 텍스트의 나열이 아니라 체계적인 구조를 가진다는 것을 느낄 수 있었다.

자기소개 페이지를 만들면서 CSS를 함께 적용해보며 디자인의 중요성도 체감할 수 있었다. 같은 HTML 구조라도 스타일을 어떻게 적용하느냐에 따라 전혀 다른 느낌의 페이지가 만들어질 수 있었고, 이를 통해 웹 개발에서 구조와 디자인이 서로 밀접하게 연결되어 있다는 점을 알게 되었다. 또한 버튼을 클릭하면 다른 페이지로 이동하도록 하는 기능을 구현하면서, 단순한 정적 페이지를 넘어 사용자와 상호작용하는 요소를 만드는 방법도 경험할 수 있었다.

이번 학습을 통해 HTML이 단순한 마크업 언어가 아니라 웹의 기본이 되는 중요한 기술이라는 것을 깨달았다. 앞으로 더 다양한 태그와 속성을 배우고, JavaScript와 같은 다른 기술과 함께 활용한다면 더욱 완성도 높은 웹 페이지를 만들 수 있을 것이라는 기대도 생겼다. 전체적으로 이번 경험은 웹 개발의 기초를 다지는 데 매우 의미 있고 유익한 시간이었다.
