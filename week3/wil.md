이번 주에는 React를 활용해 컴포넌트 구조와 상태 관리에 대해 학습했다. 특히 props와 useState를 사용하며 데이터 흐름을 이해했고, 이벤트 처리 방식에도 익숙해졌다. 또한 public 폴더 이미지 경로 문제를 겪으며 절대경로의 중요성을 알게 되었고, import 방식이 더 안정적이라는 것도 배웠다. 앞으로는 코드 구조를 더 체계적으로 설계하는 연습이 필요하다. 또한 블로그 만들기를 통해서 component 활용법에 대해서 익히게 됐다. 블로그를 만들 때 이번에는 디자인과 기능에 신경을 썼는데. 리액트 공식 사이트에서 useRef 활용법에 대해서 익힌 후에 직접 react hooks 활용법을 익혀 사용하니 뿌듯한 기분이 들었다. 기존 html과 js만 이용해도 충분히 훌룡한 웹 페이지를 만들수 있지만 react를 이용하면 시간단축과 협업이 더 편해진다는 것을 알게 돼었다. 앞으로 리액트로 프론트를 할 뿐만 아니라 디자인과 백엔드도 연습해서 풀스택 개발자가 되고 싶다.
